> [!IMPORTANT]
> **This repository is no longer maintained.**
>    
> As Minecraft continues to evolve rapidly, the architecture of the original plugin has become increasingly difficult to maintain and extend. Consequently, I have performed a complete bottom-up refactoring using Kotlin and established **FakePlayerPlus** (https://github.com/xiplugin/FakePlayerPlus) as the successor. This new version features a highly decoupled architecture, representing a more modern, robust, and enhanced iteration that will receive continued support and updates.
>   
> **本仓库已停止维护。**   
>   
> 随着 Minecraft 版本的快速迭代，原版插件的架构在修复和维护上略显吃力，故基于 Kotlin 对其进行了完全的底层重构并立项为 **FakePlayerPlus** (https://github.com/xiplugin/FakePlayerPlus) 该版本实现了项目架构的高度解耦，是更现代化、更健壮的衍生加强版，且将持续获得技术支持。
